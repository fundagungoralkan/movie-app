# 🎬 MOVIEW - 3D Cinematic Experience

This project is a **3D Movie Discovery Platform** that showcases trending films with a futuristic and aesthetic interface.
It is built using **HTML**, **CSS**, and **GSAP (GreenSock)**, featuring real-time data integration with the **TMDB API**.

# ✨ Features

**3D Circular Slider:** Interactive movie cards with depth and rotation.

**Aesthetic Focus & Blur:** Background blurs and cards center smoothly when selected.

**TMDB API Integration:** Fetches live movie data, posters, and descriptions.

**Responsive UI:** A modern, 2026-themed light mode design using "Soft Tech" aesthetics.

**Smart Centering:** Cards dynamically calculate the screen center for a perfect focal point.

**Active Search Engine:** Real-time movie search functionality.

# 📚 Technologies Used

HTML5

CSS3 (3D Transforms & Perspective)

JavaScript (ES6+)

GSAP (High-performance animations)

TMDB API

Google Fonts (Plus Jakarta Sans)
